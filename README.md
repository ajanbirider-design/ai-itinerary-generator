## 🌌 About The Project

✨ **Ajnabi AI Travel Planner** is an AI-powered web app that generates smart travel itineraries based on your:

- 💰 Budget  
- 📅 Number of days  
- 🏍️ Travel mode (Bike / Car)  

It also includes:

- 🌦️ Real-time weather insights  
- 🗺️ Google Maps route navigation  
- 🎒 Smart packing suggestions
